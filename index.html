Embed snippets

For social previews that actually render. Copy, upload PNG, go live.

Set once in <head>. The og:image URL is the single source of truth for all platforms.

Where to upload your exported PNG:
• Static / Vercel / Netlify: /public/cards/ or /og/
• Next.js app dir: public/cards/ (references /cards/gloss.png)
• Plain HTML: cards/ folder next to index.html
• CDN: any publicly-reachable path with a stable filename

Keep filenames stable for cache behaviour. Re-validate in each platform’s debugger after deploy.
